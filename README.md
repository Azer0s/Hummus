## Commands

### Variable assignment
```r
name:=value
```
### Function assignment
```r
name:=(arguments - comma separated).(process)
```
### Use function
```r
name(arguments - comma seperated)
```
### Examples
```r
x:=2
y:=(x).(x*x)
z:=y(y(x))

a:=(x,y).(x)
b:=a(true,false)
```
