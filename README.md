## Commands

### Variable assignment
```r
name:=value
```
### Function assignment
```r
name:=(arguments - comma separated).(process)
```
### Anonymous function
```r
(arguments - comma separated).(process).(values - comma separated)
```
### Use function
```r
name(arguments - comma seperated)
```
### Examples
```r
x:=2
y:=(x).(x*x)
z:=y(y(x))

a:=(x,y).(x)
b:=a(true,false)
```
